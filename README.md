# Modeling-of-Bifacial-and-Monofacial-Solar-Panels
This project develops mathematical models to evaluate how irradiance, albedo, and tilt angle affect the performance of bifacial and monofacial solar panels, validated against real installed panel data
##Methodology
- Developed MATLAB-based mathematical models for both panel types
- Installed, tested, and monitored physical panels to collect experimental performance data
- Analyzed the influence of irradiance, ground albedo, and tilt angle on energy yield and efficiency
- Compared simulation results against experimental data to assess bifacial performance gains
